How to Recover Password from a Saved Connection in DBeaver

Requirements

  • OpenSSL is installed on your system
  • OpenSSL is registered to PATH

TLDR;

cd credentials_config_json_directory # change this to the directory where credentials-config.json is located on your system
openssl aes-128-cbc -d -K babb4a9f774ab853c96c2d653dfe544a -iv 00000000000000000000000000000000 -in "credentials-config.json"

Export DBeaver Project

DBeaver stores an encrypted credentials of your saved connections in the a file named “credentials-config.json”.

To get this file we need to:

  1. In DBeaver, Go to File -> Export -> DBeaver -> Project
  2. Select an output directory
  3. Press Finish
  4. Go to output directory
  5. Rename output file and change extension to .zip
  6. Unzip it

credentials-config.json can be found on:

directory_where_you_unzipped/projects/General/.dbeaver

Decrypt credentials-config.json using OpenSSL

First let’s go to the same directory as credentials-config.json

cd directory_where_you_unzipped/projects/General/.dbeaver

Then decrypt the file using OpenSSL

openssl aes-128-cbc -d -K babb4a9f774ab853c96c2d653dfe544a -iv 00000000000000000000000000000000 -in "credentials-config.json"

It is also possible to find this file without exporting but since its location is different for each OS, I find this approach easier.
